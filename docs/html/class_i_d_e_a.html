Detailed Description

IDEA-EAX Algorithm Encryptor/Decryptor.

IDEA is the International Data Encryption Standard by Massey and Lai. The cipher is a 64-bit block cipher which uses 126-bit keys and a 64-bit initialization vector (IV). IDEA is a popular choice in open source projects such as OpenPGP.
